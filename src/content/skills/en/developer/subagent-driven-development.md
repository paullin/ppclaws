## Introduction

Use when executing implementation plans with independent tasks in the current session. Dispatches subagents per task with code review between tasks.

## Features

- Dispatches independent subagents for specific tasks
- Manages execution of multi-step implementation plans
- Enforces code review between individual tasks
- Maintains context and isolation across parallel development streams

## Applicability

Best for complex features that can be broken down into discrete, independent subtasks requiring focused execution.

## Highlights

Introduces a sophisticated delegation model where the main agent orchestrates subagents, drastically reducing context loss and hallucination.

## Usage

Use this skill when you have a multi-step plan and want the AI to handle each step independently with mandatory reviews in between.
