## Introduction

Use when creating new skills, editing existing skills, or verifying skills work before deployment.

## Features

- Scaffolds and generates new AI skills
- Edits and refines existing skill prompts
- Verifies skill functionality and syntax
- Ensures skills are deployable and well-documented

## Applicability

Designed for AI engineers and prompt designers who are extending the capabilities of AI assistants like Claude Code or Trae.

## Highlights

Provides meta-cognitive assistance: an AI skill that helps you write better AI skills with proper formatting and triggering conditions.

## Usage

Call this skill when you want to add a new capability to your AI assistant, or when you need to debug a custom skill that isn't triggering correctly.
