## Introduction

When the user wants to optimize post-signup onboarding flows, user activation, first-run experience, or time-to-value. Also use when the user mentions "onboarding flow," "activation rate," "user activation," "first-run experience," "empty states," "onboarding checklist," "aha moment," "new user experience," "users aren't activating," "nobody completes setup," "low activation rate," "users sign up but don't use the product," "time to value," or "first session experience." Use this whenever users are signing up but not sticking around. For signup/registration optimization, see signup-flow-cro. For ongoing email sequences, see email-sequence.

## Features

- Optimizes post-signup onboarding flows, first-run experiences, and time-to-value.
- Diagnoses user drop-off reasons and improves activation rates.
- Designs behavioral pathways for new users to reach their "Aha Moment".
- Applies the "Do, Don't Show" principle for interactive onboarding.

## Highlights

- Reduces user cognitive load by focusing on "one goal per session".
- Boosts user motivation through clear progress indicators and milestone celebrations.
- Differentiates core value propositions (B2B vs. B2C) to provide personalized onboarding strategies.
