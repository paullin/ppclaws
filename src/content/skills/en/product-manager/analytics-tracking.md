## Introduction

When the user wants to set up, improve, or audit analytics tracking and measurement. Use this whenever someone asks how to know if something is working or wants to measure marketing results.

## Features

- Sets up and audits analytics tracking (GA4, Mixpanel, Segment)
- Helps implement conversion and event tracking
- Assists with UTM parameters and tracking plans
- Troubleshoots tag manager and tracking issues

## Applicability

Suitable for teams needing to measure user behavior, track conversions, or debug why analytics events are not firing correctly.

## Highlights

Focuses on comprehensive measurement strategies, from defining what to track to actual implementation and troubleshooting.

## Usage

Use this skill when asking 'how do I measure this', setting up Google Analytics, or creating a tracking plan for a new feature.
