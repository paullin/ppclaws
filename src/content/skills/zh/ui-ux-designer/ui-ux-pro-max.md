## 简介

适用于 Web 和移动设备的 UI/UX 设计智能。包含 50 多种样式、161 种调色板、57 种字体配对和跨 10 个技术栈的 99 条 UX 指南。

## 功能

- 提供跨 10 个技术栈（React、Vue、SwiftUI 等）的专业知识
- 提供 50 多种 UI 样式（玻璃拟态、粗野主义、便当网格等）
- 应用 99 条 UX 指南以优化用户流程
- 包含 161 种调色板和 57 种字体配对

## 适用

任何设计跨平台应用程序团队的终极助手，提供关于 UI/UX 趋势和最佳实践的百科全书式知识。

## 亮点

无与伦比的多功能性，能够指导从 iOS 原生应用程序到基于 Tailwind 的 SaaS 仪表板的设计选择。

## 使用

调用此技能来规划 UI、审查现有代码以进行 UX 改进，或请求特定的美学风格，例如“新拟态”。
