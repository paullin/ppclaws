## 简介

令牌架构、组件规范和幻灯片生成。三层令牌（原始→语义→组件）、CSS 变量、间距/排版比例。

## 功能

- 构建强大的令牌架构（原始、语义、组件）
- 生成 CSS 变量和间距/排版比例
- 创建详细的组件规范
- 确保系统化的设计实施

## 适用

对于从头开始构建可扩展、可维护的设计系统的 UI/UX 团队和前端开发人员至关重要。

## 亮点

实施专业的三层令牌方法，弥合 Figma 设计与代码实现之间的差距。

## 使用

当您需要定义设计令牌、创建新的设计系统或跨项目标准化 CSS 变量时调用此技能。
