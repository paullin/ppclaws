## 简介

创建具有 Chart.js、设计令牌、响应式布局、文案公式和情境幻灯片策略的战略性 HTML 演示文稿。

## 功能

- 构建基于 HTML 的演示文稿和商业计划书
- 集成 Chart.js 进行数据可视化
- 应用设计令牌和响应式布局
- 利用经过验证的文案公式编写幻灯片内容

## 适用

非常适合需要快速生成专业、数据丰富的演示文稿的创始人、销售团队和营销人员。

## 亮点

通过生成由真实设计令牌驱动的响应式交互式 HTML 幻灯片，超越了静态图像。

## 使用

调用此技能来构建演示文稿，指定您的主题、数据点和所需的战略叙事。
