# 详情页与目录页设计还原描述

本文档根据 [设计还原](../prompt/设计还原.md) 的要求，对高保真原型页面（详情页 `prd-automation-assistant-cn.html` 与目录页 `skills-for-product-manager-cn.html`）进行像素级、100% 细节拆解描述。

## 1. 整体布局与栅格系统
- **页面结构**：顶部固定导航栏（Header） + 居中主内容区（Main） + 底部页脚（Footer）。
- **最大宽度与栅格**：主内容区最大宽度为 `max-w-4xl`（896px），水平居中 `mx-auto`，左右内边距 `px-4 sm:px-6 lg:px-8`，上下外边距 `py-8`。
- **背景与文字颜色**：
  - 浅色模式：背景 `#f9fafb`（bg-gray-50），文字 `#111827`（text-gray-900）。
  - 深色模式：背景 `#0d0f14`（bg-astro-dark），文字 `#f3f4f6`（text-gray-100）。
- **组件垂直间距**：主内容区各主要模块之间使用 `space-y-12`（48px 间距）隔开。

## 2. Breadcrumbs（面包屑导航）
- **位置**：Main 内容区顶部。
- **排版与间距**：Flex 布局，水平垂直居中对齐 `items-center`，元素间距 `gap-2`（8px）。
- **字体与颜色**：
  - 基础文字：`text-sm`（14px），默认颜色 `#6b7280`（text-gray-500），深色模式 `#9ca3af`（dark:text-gray-400）。
  - 可点击链接（a标签）：Hover 时文字颜色变为 `#8b5cf6`（text-astro-accent），带 `transition-colors` 过渡效果。
  - 当前页面文字：字体加粗 `font-medium`（字重 500），颜色 `#111827`（text-gray-900），深色模式 `#ffffff`（dark:text-white）。
- **图标**：`chevron-right`，宽高 `w-4 h-4`（16x16px）。

## 3. ArticleHeader（详情页顶部区域）
- **间距包裹**：使用 `space-y-8`（32px）包含大图与简介。
- **大图模块**：
  - 尺寸与圆角：相对定位 `relative`，比例 `aspect-video`（16:9），宽度 `w-full`（100%），圆角 `rounded-2xl`（16px），溢出隐藏 `overflow-hidden`。
  - 边框与阴影：边框宽度 1px，颜色 `#e5e7eb`（border-gray-200），深色模式 `#2e364f`（dark:border-astro-border），阴影 `shadow-lg`。
  - 标签（Badge）：绝对定位 `absolute top-4 left-4`（左上角各16px），内边距 `px-3 py-1`，背景 `#8b5cf6`（bg-astro-accent），文字白色，字号 `text-xs`（12px），字重 `font-bold`，圆角 `rounded-full`，带阴影 `shadow-lg`。
- **简介区域**：包裹在 `space-y-4`（16px 间距）中。
  - **大标题 (H1)**：字号 `text-4xl sm:text-5xl`（移动端 36px，PC 端 48px），字重 `font-extrabold`（800），字间距 `tracking-tight`。
  - **元数据（Metadata）**：Flex 布局，元素间距 `gap-4`，字号 `text-sm`，颜色 `#6b7280`（深色 `#9ca3af`）。每个子项内部 `flex items-center gap-1.5`（图标与文字间距 6px）。图标尺寸 `w-4 h-4`。
  - **摘要描述 (p)**：字号 `text-xl`（20px），颜色 `#4b5563`（text-gray-600，深色模式 text-gray-400），行高 `leading-relaxed`（1.625）。

## 4. 全局 Markdown 样式（.prose）
- **分隔线**：在文章标题与正文之间有 `<hr>`，边框颜色 `#e5e7eb`（深色 `#2e364f`）。
- **主体容器**：`<article class="prose dark:prose-invert max-w-none">`。
- **H2 标题**：需通过全局样式覆盖。字号 `text-2xl`（24px），字重 `font-bold`（700），上外边距 `mt-10` / `margin-top: 2rem`，下外边距 `margin-bottom: 1rem`。
- **段落 (p)**：下边距 `margin-bottom: 1rem`，行高 `line-height: 1.7`，文字颜色 `#4b5563`（深色模式 `#9ca3af`）。
- **无序列表 (ul/li)**：类型为实心圆点 `list-disc`，左内边距 `pl-6` / `padding-left: 1.5rem`，列表间距 `space-y-2`。每个 li 下边距 `0.5rem`。
- **加粗 (strong)**：字重加粗。

## 5. RelatedLinks（关联资源列表）
- **位置与容器**：上内边距 `pt-8`（32px），顶部边框 1px 颜色 `#e5e7eb`（深色 `#2e364f`），内部元素间距 `space-y-4`。
- **模块标题 (H2)**：字号 `text-2xl`（24px），字重 `font-bold`，字间距 `tracking-tight`，文字颜色 `#111827`（深色 `#ffffff`）。
- **链接列表**：Flex 竖向排列 `flex-col gap-2`（8px）。
- **单个链接 (a)**：文字颜色 `#111827`（深色 `#d1d5db`），字号 `text-sm`，Hover 状态颜色 `#8b5cf6`（text-astro-accent）并带有下划线 `underline`，`transition-colors` 动画。

## 6. ExploreMore（探索其他分类模块）
- **位置与容器**：上内边距 `pt-16`（64px），顶部边框 1px 颜色 `#e5e7eb`（深色 `#2e364f`）。
- **模块标题 (H2)**：字号 `text-2xl`，字重 `font-bold`，字间距 `tracking-tight`，下外边距 `mb-6`（24px）。
- **卡片网格**：Grid 布局，移动端 `grid-cols-1`，平板及以上 `sm:grid-cols-3`，网格间距 `gap-6`（24px）。
- **单个卡片 (a 标签, group)**：
  - 布局与间距：Flex 布局，水平居中对齐 `items-center`，内部间距 `gap-4`（16px），内边距 `p-5`（20px）。
  - 外观：圆角 `rounded-xl`（12px），边框 1px `#e5e7eb`（深色 `#2e364f`），背景色 `#ffffff`（深色 `#1e2330` bg-astro-card）。
  - 交互状态：Hover 时边框变色为 `#8b5cf6`，增加阴影 `hover:shadow-lg`，`transition-all`。
  - **左侧图标区块**：宽高 `w-12 h-12`，圆角 `rounded-lg`，背景色 `#8b5cf6` 透明度 10%（bg-astro-accent/10），图标颜色 `#8b5cf6`，居中显示。图标尺寸 `w-6 h-6`。Hover 时放大 `group-hover:scale-110`。
  - **中间文本区块**：Flex 占比 `flex-1`。标题字号 `text-base`，字重 `font-bold`，Hover 时颜色变 `#8b5cf6`。副标题字号 `text-xs`，颜色 `#6b7280`（深色 `#9ca3af`），上外边距 `mt-1`。
  - **右侧箭头图标**：`chevron-right`，尺寸 `w-5 h-5`，默认颜色 `#9ca3af`，Hover 时颜色变 `#8b5cf6` 并向右平移 `group-hover:translate-x-1`，带 `transition-transform` 过渡。
