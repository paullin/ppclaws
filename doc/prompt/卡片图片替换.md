# 卡片图片替换

## 背景

我们已经生成了新的首页、目录页、详情页。这些页面上都有大量的图文卡片。

目前这些卡片的图片都是通过在线展位图服务服务提供的。图片与文字无关。

未来上线后，我们需要使图文相关。

## 任务

我担心作为一个网站，如果有太多图片，会增加加载时间，导致网站性能不佳，从而影响 SEO 排名。

但如果我不使用图片（不管是本地图片还是在线图片），还有什么方式来让整个网站的图文卡片看起来更专业？更美观呢？

我希望你帮我思考可行的建议方案。

## 要求

- 输出：将建议方案输出到 [卡片图片替换方案](../tech/卡片图片替换方案.md)
- 性能第一、SEO 效果第一、美观第二
