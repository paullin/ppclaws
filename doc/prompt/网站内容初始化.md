# 网站内容初始化

下面是一系列用于产品经理、UI/UX设计师、市场营销、开发人员的知名 技能列表。

## 背景

我们已经完成了 PPClaws.com 网站的搭建和基本配置。接下来我们需要开始为网站添加冷启动内容。

## 任务

接下来你将需要阅读 [skills_list.xlsx](../skills_list.xlsx) 文件，并执行如下任务：

- Step1：阅读 skills_list.xlsx 文件，了解每个字段的含义
- Step2：根据 Excel 文件构建对应的目录树（首页->二级分类->三级分类->技能详情页）
- Step3：为每一层级的页面，创建对应的 JSON 配置文件，用于配置该页面的元数据和内容
- Step4：为每个技能创建对应的 Markdown 文件，用于展示该技能的详细信息
- Step5：运行 npm run build，生成静态网站文件
- Step6：运行 npm run dev，启动开发服务器
