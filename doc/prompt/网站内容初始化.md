# 网站内容初始化

## 背景

我们已经完成了 PPClaws.com 网站的搭建和基本配置。接下来我们需要开始为网站添加冷启动内容。

当网站初始化内容被正确添加后，我们将可以看到正确的页面。

## 任务

接下来你将需要阅读 [skills_260406.md](../skills/skills_260406.md) 文件，并执行如下任务：

- Step1：阅读 skills_260406.md 文件，了解每个 skills 的基本情况
- Step2：根据 Markdown 文件构建对应的目录树（首页->二级分类->三级分类->技能详情页）
- Step3：为每一层级的页面，创建对应的 JSON 配置文件，用于配置该页面的元数据和内容
- Step4：为每个技能创建对应的 Markdown 文件，用于展示该技能的详细信息
- Step5：运行 npm run build，生成静态网站文件
- Step6：运行 npm run dev，启动开发服务器

## 要求

我们需要持续添加新的技能，所以我们的网站必须要符合以下要求：

- 所有的 Skills 都按照添加日期的顺序进行展示。越新添加的 Skills 越靠前。
- 如果某个技能在 Excel 文件中被标识为「精选」，则该技能对应的卡片会出现在首页和 Skills 的精选区域。但依然遵循按时间顺序展示的原则。
