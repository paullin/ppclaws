# 设计还原

我现在需要你对即将作为参考的高保真原型页面做**像素级、100%细节还原**的完整描述。
要求：

1. 严格按照从上到下、从左到右的顺序拆解每一个元素，不漏任何细节。
2. 对每一个元素描述必须包含：
   - 位置（顶部/左侧/居中/右侧/底部）
   - 尺寸比例（宽度占比、高度、间距）
   - 颜色（背景色、文字色、边框色、阴影色）
   - 字体（字号、字重、行高、对齐方式）
   - 间距（内边距 padding、外边距 margin、元素之间 gap）
   - 圆角、边框、阴影、透明度、渐变
   - 图标/图片大小、位置、比例
   - 交互状态（默认/hover/点击/禁用）
3. 对整体布局描述：
   - 页面结构（头部、导航、内容区、侧边栏、底部）
   - 栅格系统（几列、最大宽度、居中方式）
   - 响应式规则（PC / 移动端表现）
4. 对文本内容逐行还原，包括大小写、标点、空格、换行。
5. 对所有可点击区域明确标注：按钮、链接、卡片、输入框、下拉框等。
6. 所有的二、三级目录页面，主标题与副标题必须统一为“居中对齐（text-center）”，且其容器必须包含 `pt-16 pb-8` 从而留出足够的顶部呼吸空间，并且不再使用顶部面包屑导航（由于头部已有导航菜单）。
7. 禁止概括、禁止模糊描述、禁止省略细节。
8. 最终输出结构清晰、层级分明，前端工程师可以直接根据你的描述还原出像素级一致的页面。
