# 首页替换

## 背景

我已经在 [英文原型页面](../prototype/index-en.html)和 [中文原型页面](../prototype/index.html)中实现了原型页面。

这个页面是用 HTML + Tailwind CSS + JS 实现的，没有用任何框架。

现在我要基于这个原型页面来替换现在主题的首页。

## 任务

请帮我分析如果基于原型页面来替换现在主题的首页，需要做哪些改动？

- 改动后的首页需要和原型页面保持百分百像素级一致。
- 改动后的首页使用的技术栈需要和当前网站使用的技术栈一致。
- 改动后的首页所有板块需要可以通过运营后台配置出来。（板块的数量、文案、图片）
- 改动后的网站支持存储用户提交的订阅邮箱、收录信息。
- 改动后的网站需要支持多语言 / 搜索 / 暗夜模式。
- 改动后的网站需要支持响应式布局。
